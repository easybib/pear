easybib's PEAR Channel Server

Registering the channel: 
pear channel-discover easybib.github.com/pear
Listing available packages: 
pear remote-list -c easybib
Installing a package: 
pear install easybib/package_name
Installing a specific version/stability: 
pear install easybib/package_name-1.0.0
pear install easybib/package_name-beta
Receiving updates via a feed: 
http://easybib.github.com/pear/feed.xml
